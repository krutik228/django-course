# **Learning the framework**


## Django course from: https://www.youtube.com/user/yuriikozlov , thanks to the author

### The news site is used as an example. The user has the ability to view news by category, register, log in, and then add his own news.The admin site has an advanced text editing panel.The site has pagination and highlighting the current category. Notify the user of login or registration errors. Also, an unauthorized user cannot add their own news. Customized a nice gray theme for the administration site. Implemented captcha for registration.
